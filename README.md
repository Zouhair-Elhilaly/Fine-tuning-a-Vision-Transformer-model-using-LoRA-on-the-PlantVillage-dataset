# Fine-tuning Vision Transformer with LoRA on PlantVillage Dataset

## 📌 Project Overview

This project demonstrates how to fine-tune a **Vision Transformer (ViT)** model using **LoRA (Low-Rank Adaptation)** for **plant disease classification**.

The model is trained on the **PlantVillage dataset**, which contains images of plant leaves with different diseases. The goal is to classify the disease from leaf images using a parameter-efficient fine-tuning method.

---

## 📊 Dataset

The **PlantVillage dataset** contains thousands of labeled images of healthy and diseased plant leaves. It is commonly used for plant disease detection tasks in computer vision.

---

## 🧠 Model

* Vision Transformer (ViT)
* LoRA (Low-Rank Adaptation) for efficient fine-tuning

LoRA allows fine-tuning large models by updating only a small number of parameters instead of the full model.
